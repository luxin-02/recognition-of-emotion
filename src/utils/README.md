# 详细说明

## js
- array.js 输出为数组的方法
  1. 数组对象去重
  2. 从数组中随机选取多个元素

- export.js 导出模板文档
  1. 根据模板导出Word文件(单个)
  2. 根据模板导出Word文件，并打包成zip(多个)
   
- flexible.js 自适应窗口

- number.js 关于输出类型为number的方法
  1. 范围之间的随机数
  2. px转rem
  3. rem转px
  4. 缩放比列

- object.js 对象方法
  1. 深复制
  2. 将第二个对象的非空属性拷贝给第一个对象所拥有的非空属性
  3. 通过data对象快速生成FormData对象
  4. 遍历对象 如果属性值是字符串类型，则去掉左右空格

- time.js 关于日期、时间、生日相关的方法
  1. 时间戳转时间格式
  2. 根据生日计算当前年龄
  3. 根据年龄计算大概生日(1月1号)
  4. 数字转时分秒/计时
  5. 获取时间戳(默认10位)

- url.js 关于路径相关的方法
  1. 图片文件转base64
  2. 将base64转换为blob
  3. 将blob转换为file
  4. 将base64转换为file
  5. 下载Base64文件
  6. 下载文件

- type.js 判断数据类型
  1. 判断是否为String
  2. 判断是否为Array
  3. 判断是否为File
  4. 判断是否为Null
  5. 判断是否为Base64
  6. 判断是否为Date

- directive.js 自定义指令
  1. 权限指令注册
  2. 弹窗移动指令
  3. 防抖指令
  4. 节流指令

- filters.js 自定义过滤器
  1. 时间日期格式化
  2. 将秒数按格式转换为时分秒

# 游客首页部门字段防护设计

## 问题

游客登录返回的 `userInfo` 不包含 `depts` 字段。首页模板直接读取
`$store.getters.userInfo.depts.length`，导致 Vue 渲染抛出 `TypeError`，页面白屏。

## 设计

仅修改 `src/views/front/index/Home.vue` 的部门信息渲染条件：先确认 `depts`
是数组且数组非空，再显示部门。游客没有部门数据时隐藏整行；普通用户的显示行为不变。

不修改登录接口响应，不在 Vuex 中补充默认字段，也不调整其他页面。

## 验证

- 游客登录进入首页时不再出现读取 `length` 的渲染错误，且不显示部门行。
- 普通用户存在部门数据时仍显示第一个部门名称。
- 项目构建通过。

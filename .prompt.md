# 微笑打卡四状态管理

## 任务目标
管理 `shoot.vue` 组件的四个状态流转，控制UI显示、提示文本和按钮文字的动态变化。

## 四个状态定义

```
shooting  → 拍摄中     | paishe_zhon    | 提示：请在光线充足的地方进行检测，避免曝光或背光。
shot      → 拍摄后     | paishe_hou     | 提示：请在光线充足的地方进行检测，避免曝光或背光。
analyzing → 分析中     | paishe_fenxizhon | 提示：照片情绪分析中，请勿关闭...
analyzed  → 分析后     | paishe_fenxihou | 提示：分析完成！
```

## UI 联动映射

### 1. 状态对应的显示组件（v-if）
| 状态 | paishe_zhon | paishe_hou | paishe_fenxizhon | paishe_fenxihou |
|-----|------------|-----------|------------------|-----------------|
| shooting | true | false | false | false |
| shot | false | true | false | false |
| analyzing | false | false | true | false |
| analyzed | false | false | false | true |

### 2. 按钮文字映射
| 状态 | 左按钮 | 右按钮 |
|-----|-------|-------|
| shooting | 开始拍摄 | 取消拍摄 |
| shot | 分析照片 | 重新拍摄 |
| analyzing | 发布打卡照片 | 重新拍摄 |
| analyzed | 发布打卡照片 | 重新拍摄 |

### 3. 提示文本（tips_text）
- `shooting` / `shot`: "请在光线充足的地方进行检测，避免曝光或背光。"
- `analyzing`: "照片情绪分析中，请勿关闭..."
- `analyzed`: "分析完成！"
